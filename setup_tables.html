🚀 Supabase 表创建助手

Lumio VIP 功能 - 数据库表自动配置

⚠️ 重要：由于 Supabase 安全限制，无法通过代码直接创建表。请按照以下 3 个简单步骤操作。
1 点击按钮复制 SQL 语句
SQL 语句已准备好，点击下方按钮自动复制到剪贴板。
2 打开 Supabase SQL Editor
点击下方按钮打开 Supabase 控制台，然后点击 "New query"。
3 粘贴并执行
在 SQL Editor 中按 Cmd+V 粘贴，然后点击 "Run" 按钮执行。
🔗 打开 Supabase SQL Editor
✅ SQL 已复制到剪贴板！现在请打开 Supabase SQL Editor 并粘贴（Cmd+V）。

📖 执行后的效果

创建以下表： ✅ usage_counts - 用户使用次数和 VIP 状态 ✅ purchases - 购买记录 包含： • 自动生成 UUID • RLS 安全策略 • 索引优化 • 自动更新时间戳触发器
